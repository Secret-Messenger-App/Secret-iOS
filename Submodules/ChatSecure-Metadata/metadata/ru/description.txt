ChatSecure - это бесплатное приложение для общения с открытым исходным кодом, которое поддерживает протокол OTR над XMPP. Вы можете подключить существующий Google-аккаунт, создать учетную запись на любом публичном XMPP-сервере (в том числе через Tor) или вообще воспользоваться собственным сервером для дополнительной безопасности.

В отличие от программ, ограничивающих вас собственным сервисом, ChatSecure полностью совместим с другими приложениями, поддерживающими OTR и XMPP, такими как Adium, Jitsi и множеством других.

ChatSecure использует только зарекомендовавшие себя криптографические библиотеки с открытым исходным кодом для защиты вашей переписки. Другие приложения могут заявлять о шифровании "военного уровня", но без публичного аудита исходного кода и поддающегося проверке сквозного шифрования вы не можете быть точно уверены в безопасности.

* XMPP с привязкой TLS-сертификата
* OTR для поддающегося проверке сквозного шифрования и секретной пересылки.
* OMEMO в качестве адаптированной для мобильных устройств альтернативы OTR.
* Интеграция с Tor для легкого доступа к .onion XMPP-сервисам.
* SQLCipher для локального шифрования истории переписки.
* Свободное ПО: https://github.com/chatsecure/chatsecure-ios
